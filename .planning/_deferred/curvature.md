# Deferred: Curvature Estimation from Path Data

With enough waypoint paths, estimate sectional curvatures of behaviorally-revealed conceptual space.

- Embedding analysis suggests hyperbolic geometry (Nickel & Kiela, HELM)
- But behavioral navigation might reveal different curvature than static embeddings
- Multiple paths between the same points at different resolutions (3 vs 5 vs 10 waypoints) could reveal scale-dependent curvature
- Riemannian framework from Arvanitidis/Hauberg provides the math

Requires: substantial path dataset (probably post-phase 3), careful distance metric selection, potentially Python for differential geometry computations.
